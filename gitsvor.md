1. Hvað gera eftirfarandi Linux skipanir?

   * cd: Breytir vinnumöppunni
   * ls: Sýnir lista af skrám og möppum í möppu
   * pwd: Sýnir vinnumöppu
   * mkdir: Býr til möppu

2. Hvað gera eftirfarandi Git skipanir?

   * git clone: Klónar Git repo
   * git status: Sýnir allt sem hefur ekki verið pushað
   * git diff: Sýnir munin frá síðasta commit

3. Hvað gera eftirfarandi Git skipanir saman? Hvaða gagn er að þeirri aðgerð?

   * git log: Sýnir fyrrverandi commit 
   * git checkout: Sýnir skrár í valdri grein
   * git branch: Sýnir greinar
